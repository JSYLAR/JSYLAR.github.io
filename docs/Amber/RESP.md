Amber中对小分子处理不包括结构优化与RESP电荷的计算，仅计算bcc电荷，完全够用但相对准确性较低，记录一下使用G16优化小分子结构并计算RESP的流程

#### 一、小分子准备
小分子可以是新绘制的或已有的，格式为X.pdb或X.mol2
####  二、转换为高斯输入格式
使用Gaussview打开小分子结构文件，保存为X.gjf格式

#### 三、运行高斯

```
g16<X.gjf>X.out
```
得到输出结果`X.out`
####  四、将优化结果转化为.mol2文件

```
antechamber -i X.out -fi gout -o X.mol2 -fo mol2 -c resp -at amber
```
